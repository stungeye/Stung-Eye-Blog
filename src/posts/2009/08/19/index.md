[Fundamental Principles of Cognition](http://www.foundalis.com/res/poc/PrinciplesOfCognition.htm)

<p><em>&#8220;If anyone makes a claim of having constructed (programmed) a cognitive agent, that agent should show evidence of adhering to [these 6½ Fundamental Principles]. I submit that the fewer of these principles an agent employs, the less cognitively interesting the agent is.&#8221;</em></p>

<ul>
<li>Principle 1: <a href="http://www.foundalis.com/res/poc/PrinciplesOfCognition.htm#first_law">Object Identification</a> (Categorization)</li>
<li>Principle 2: <a href="http://www.foundalis.com/res/poc/PrinciplesOfCognition.htm#second_law">Minimal Parsing</a> (“Occam’s Razor”)</li>
<li>Principle 3: <a href="http://www.foundalis.com/res/poc/PrinciplesOfCognition.htm#third_law">Object Prediction</a> (Pattern Completion)</li>
<li>Principle 4: <a href="http://www.foundalis.com/res/poc/PrinciplesOfCognition.htm#fourth_law">Essence Distillation</a> (Analogy Making)</li>
<li>Principle 5: <a href="http://www.foundalis.com/res/poc/PrinciplesOfCognition.htm#fifth_law">Quantity Estimation and Comparison</a> (Numerosity Perception)</li>
<li>Principle 6: <a href="http://www.foundalis.com/res/poc/PrinciplesOfCognition.htm#sixth_law">Association-Building by Co-occurrence</a> (Hebbian Learning)</li>
<li>Principle 6½: <a href="http://www.foundalis.com/res/poc/PrinciplesOfCognition.htm#six_and_a_half_law">Temporal Fading of Rarity</a> (Learning by Forgetting)</li>
</ul>
