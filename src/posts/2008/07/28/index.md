<p>Stung Eye is born again. This is version 3, a lifestream.</p>

<p>This site is now continually (and automatically) generated as I interact with various web services.</p>

<ul>
<li>To share a link I <a href="http://del.icio.us/stungeye/tumblr_links">tag it</a> in <a href="http://del.icio.us/stungeye/">my del.icio.us linkstream</a>.</li>
<li>To share a photo I <a href="http://flickr.com/photos/stungeye/tags/tumblr/">tag it</a> in <a href="http://">my flickr photostream</a>.</li>
<li>To post a blog entry I use <a href="http://www.tumblr.com">Tumblr.</a>
</li>
<li>Comments are handled by <a href="http://disqus.com/">Disqus</a> which in turn provides a <a href="http://stungeye.disqus.com/">commentstream</a>.</li>
</ul>
<p>I feel so… aggregated.</p>

<p>The site runs on a custom platform that uses <a href="http://codeigniter.com/">CodeIgniter</a> and the <a href="http://www.tumblr.com/api">Tumblr API</a>. The code is based on a <a href="http://www.yongfook.com/post/view/92/how-to-make-your-own-lifestream-blog">pre-release version of SweetCron</a>. Currently the theme is a slightly tweaked version of the default Tumblr theme. This will change with time.</p>

<p>Various flavours of RSS feeds are available:</p>

<ul>
<li><a href="http://stungeye.com/feed">All Content RSS</a></li>
<li><a href="http://stungeye.com/feed/blog">Blog Only RSS</a></li>
<li><a href="http://stungeye.com/feed/photo">Photo Only RSS</a></li>
<li><a href="http://stungeye.com/feed/link">Link Only RSS</a></li>
</ul>
<p>The <a href="http://stungeye.com/archives/">version 2 Stung Eye Archives</a> are still available. I have plans to integrate much of the old non-blog content (<a href="http://stungeye.com/sounds/">sounds</a>, <a href="http://stungeye.com/design/">design</a>, <a href="http://stungeye.com/programs/">programs</a>, <a href="http://stungeye.com/words/">words</a>).</p>

<p>If you find any bugs, please let me know. Thank you kindly.</p>
