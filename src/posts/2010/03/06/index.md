## Mobile Development

<p>This weekend I&#8217;ll be exploring:</p>

<ul>
<li>
<a href="http://phonegap.com/">PhoneGap</a> - Cross-platform mobile app-dev using html / css / javascript.</li>
<li>
<a href="http://www.jqtouch.com/">JQTouch</a> - A mobile plugin for my fav Javascript framework, <a href="http://jquery.com">JQuery</a>.</li>
<li>
<a href="http://luckybite.com/iprocessing/">iProcessing</a> - The API from <a href="http://processingjs.org">processing.js</a> extended for mobile use.</li>
<li>
<a href="http://www.genuitec.com/mobile/features.html">MobiOne</a> - Windows mobile emulator for iPhone, Blackberry, Android and Pre.</li>
</ul>
<p><strong>Step one</strong>: <a href="http://phonegap.pbworks.com/Getting-started-with-Android-PhoneGap-in-Eclipse">Install Eclipse, PhoneGap and the Android SDK</a> along with JQTouch, iProcessing and MobiOne.</p>

<p><strong>Step two</strong>: Play around. Create a few sample apps.</p>

<p><strong>Step Three</strong>: Tomorrow Andrew and I will deploy these apps to his Android phone.</p>
