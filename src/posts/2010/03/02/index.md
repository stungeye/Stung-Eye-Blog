[Waterloo / Google AI-Tron Challenge Results](http://csclub.uwaterloo.ca/contest/rankings.php)

<p>The goal of this challenge was to write a computer-program capable of playing the game <a href="http://en.wikipedia.org/wiki/Tron_%28video_game%29">Tron</a>.</p>

<p>The rankings for <a href="http://csclub.uwaterloo.ca/contest/profile.php?user_id=1815">my Ruby bot</a>:</p>

<ul>
<li>146 / 708&#160;<a href="http://csclub.uwaterloo.ca/contest/rankings.php">Total Entries</a>
</li>
<li>3 / 23&#160;<a href="http://csclub.uwaterloo.ca/contest/language_profile.php?lang=Ruby">Ruby Entries</a>
</li>
<li>16 / 88&#160;<a href="http://csclub.uwaterloo.ca/contest/country_profile.php?country_id=2">Canadian Entries</a>
</li>

<p>The rankings were determined using tournament play along with the <a href="http://en.wikipedia.org/wiki/Elo_rating_system">Elo rating system</a>.</p>

<p>Early non-<a href="http://www.sifflez.org/misc/tronbot/index.html">minmax</a> versions of <a href="http://github.com/stungeye/Tron-Bot">my bots are available on github</a>. <a href="http://github.com/a1k0n/tronbot/">The winning C++ bot source</a> is also available; as is <a href="http://csclub.uwaterloo.ca/contest/forums/viewtopic.php?f=8&amp;t=358#p1877">a brief explanation by the author</a>.</p>

<p>I enjoyed this challenge immensely. My <a href="http://www.ruby-lang.org/en/">Ruby programming</a> skills also benefited from this <a href="http://codekata.pragprog.com/2007/01/code_kata_backg.html">Code Kata</a>.</p>
