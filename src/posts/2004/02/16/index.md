The following 'sketches' are all explorations of the <a href="http://processing.org/" target="_blank">Processing</a> language. Source code is provided for all of the applets.


The first sketch is a new take on personal horoscopes: Part <a href="http://skepdic.com/inkblot.html" target="_blank">Rorschach inkblot</a>, part <a href="http://pacificcoast.net/~wh/Index.html" target="_blank">I Ching</a>, part <a href="http://www.klov.com/S/Space_Invaders.html" target="_blank">space invaders</a>.
