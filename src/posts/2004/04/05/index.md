Our move is complete. It's official, we own too much <i>stuff</i>. However, an early summer garage-sale followed by some donations to the local Sally Ann should solve this problem.

***

As I mentioned in my last post, I've been using <a href="http://www.spurl.net/" target="_blank">Spurl</a> to catalog and <a href="http://www.stungeye.com/spurl.htm">syndicate my web surfing</a>. Since then, I've discovered some similar services. These tools provide a new framework for net-based exploration, classification and value assignment; a proto-<a href="http://www.nettuno.it/fiera/electric.italy/noosphere.html" target="_blank">noosphere</a>. 

The tools:

<li><a href="http://www.furl.net/index.jsp" target="_blank">Furl</a> (Similar name, but this one has been around for over a year.)

</li><li><a href="http://del.icio.us/" target="_blank">del.icio.us</a> (Now integrated with Spurl!)

</li><li><a href="http://www.stumbleupon.com/" target="_blank">Stumble Upon</a> (A word-of-mouth approach to websurfing.)

</li><li><a href="http://www.memestreams.net/topics/" target="_blank">MemeStreams</a> (A collaborative weblog.)

</li><li><a href="http://www.kinja.com" target="_blank">Kinja</a> (Spurl for blog posts?)


I also recommend <a href="http://perceptionalism.com/archives/000095.html" target="_blank">this post</a> over at Information Pollution. The developers of both Spurl and Furl make an appearance in the comments.
