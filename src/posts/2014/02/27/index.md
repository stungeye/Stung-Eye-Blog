<p>Lines, lines, lines. Over the past week or so I’ve been working on a computer program (see above) that can generate images similar to my dad’s engravings (see below).*</p>

<p>I’m not quite done, but the eventual goal is to transfer an image generated by this process to a copper plate by way of the <a href="http://www.slideshare.net/stungeye/etching-for-artists-and-hackers-printmaking-circuitboards">etching process</a> in order to print the image to paper.</p>

<p><em>When the above program has finished, you can click it to start the process again.</em></p>

<p>Earlier versions of this program can be seen here:</p>

<ul><li><a href="http://www.openprocessing.org/sketch/135314">First Line Intersection Test</a> (Hover mouse over image.)</li>
<li><a href="http://www.openprocessing.org/sketch/136594">Sketching Lines</a> (No circle in centre.)</li>
<li><a href="http://www.openprocessing.org/sketch/136653">Alluded Circle</a> (Similar to the above program but larger and not as smoothly animated.)</li>
</ul><p>* Truthfully I was trying to create an algorithm for something else and stumbled across one that created dad-like images.</p>
